## 🚀 快速开始

### 📦 安装插件

在使用之前，需要先全局安装 `vsext` 工具：

```
npm i -g vsext
```

### ⬇️ 下载插件

要下载当前系统中已安装的所有插件，可以运行以下命令：

```
vsext d
```

这将会将所有已安装插件的压缩包下载到当前目录。

### 📥 安装已下载的离线插件

若要安装之前已经离线下载好的插件，可以使用以下命令：

```
vsext i
```

这将会安装已下载的插件文件。
